== Installation ==

Şöylece kurulur : 

1. MahkemeKararıyla klasörünü wp-content/plugins klasörüne atın
1. wp-admin'de Eklentiler kısmından eklentiyi aktifleştirin.
1. Ayarlar > Mahkeme Kararıyla Kapat kısmından özelleştirin.
